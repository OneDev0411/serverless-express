AWS Serverless Express

Welcome to your AWS serverless Express application. This example application has several routes configured for you to explore. State is stored in memory in a given container, and thus ephemeral - see Understanding Container Reuse in AWS Lambda for more information.

Routes

GET /users

Returns a list of all users.

curl /users

POST /users

Creates a new user.

curl /users -X POST -d '{"userName":"Jake"}' --header 'content-type: application/json'

GET /users/:userId

Returns a single user.

curl /users/1

PUT /users/:userId

Updates an existing user.

curl /users/1 -X PUT -d '{"userName":"Jane"}' --header 'content-type: application/json'

DELETE /users/:userId

Deletes an existing user.

curl /users/1 -X DELETE
